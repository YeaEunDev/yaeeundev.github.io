### 연산자(Operator)

#####논리 연산자
- not : 논리 부정
- and : 논리 곱
- or : 논리 합
-우선수위가 높은 순서 = 먼저 수행되는 순서

#####비교 연산자
- == : 같다
- != : 같지않다
- " > , < , >= , <=" 
-우선수위가 높은 순서 = 먼저 수행되는 순서

#####연산자의 우선순위
1. 산술 연산자
2. 비교 연산자 & in, not in 연산자
3. 논리 연산자


##### 불리언식
: True 나 False 중에 하나의 값을 결과로 산출하는 표현식

```
a = 3
b = 5
c = a > b  # 변수 a값이 b보다 크면 True 할당
print(c)   # False값 출력
```

##### 불리언식 부정
1. 전체 식에 not() 표현하기
2. 모든 연산자를 반대 놀리로 바꿔서 표현하기
	-두가지 방법 중 하나 선택
